周报#05 Oh, unimelb
Tommy
发布于 2025/03/08

Briefing

  • 旅行计划泡汤 看到去日本的机票很便宜，想着找还在国内的同学一起去。很可惜，一共三个人，两个都没有时间，成行不了了。几个月之前开始看一个博主的美西自驾V-log，这周又看到Links去美西复刻macOS壁纸。现在只好家里蹲，等以后的机会吧。

  • Oh, unimelb 去年选专业的时候留意过Handbook，注意到数学和统计的选课可以根据自己的secondary qualification，选择性的跳过第一阶段内容的。因为是midterm-intake，选课系统还没有对我开放，就去问了Step 1能不能查到我的equivalent secondary studies。几天后回复系统里看不到，看来UNSW Foundation的成绩不能自动转换，已经提交申请。

  • 学习OpenCV OpenCV University上的免费课程，教授最基础的机器视觉处理，每一节都分为课程视频和Quiz两部分。视频不长，根据提供的Jupiter Notebook简单讲解。Quiz部分很像传说中的国外大学考试，不仅限于课程提供的内容，不少题需要自己去阅读相关文档才能做。

  • Node.js 博客用hexo构建，装修需要Node.js才行。一直它有畏难情绪，不熟悉的js语法看起来跟魔法一样，很不习惯。再加上ES6，感觉语法标准混乱，一时分不清楚，奇奇怪怪的导入外部库的方式（甚至有两种，还不兼容！），很像几年前我一直不愿意接触Python的感觉。借着这次机会，接触一下这门“未来的语言”（其实我觉得这仅限于需要敏捷开发的一些应用场景，特别是对后端，个人更倾向于Go和Rust这类语言）。目前感觉还不错，勉强可以改改博客。

  • 50岁老镜头打赢复活赛！ FD-Z转接环到货，装上80-200，边缘画质依然清晰锐利，轻微紫边。得益于新机身不错的人体工学设计，不会像在AE-1上一样需要用几根手指捏住沉重的机身。另外一颗50f1.4也很不错，变焦环非常丝滑，有要替代50f1.8D的冲动。找机会去附近玩，可以拍拍试试。

  • “云霄飞车”almost done 断断续续搭了三周，三十五米的塑料管都用完了。轨道搭建部分都基本完工，剩下的就是调试工作。这个过程就像写代码，钢珠在轨道上滚，就是程序运行；小球脱轨，就是程序报错崩溃；调整轨道，就是debug。还不会写代码的时候就喜欢这种玩具？？！原来我是先天程序员体质😦

  • 多亏我亲爱的老师

    1. 设备，合体！ 订做的东西终于都到了，程序也调好了。组装到一起，有种恍惚感，不敢相信都是我自己折腾出来的设备。从一开始的选型，到建模定制，都是全新的技能。

    2. 啊？学术会议？ 被委以重任，寻找合适的国际学术会议。怎么也没料到需要解决这件事，虽说不是我需要参加。能见识一下，顺便出国转一圈。

    3. 讲座？我喜欢 去蹭听大学校内的讲座。

    4. 哇塞，打车费 没什么好说的，四十五度，鞠躬，太感谢啦！

  • 侃哥直播 周一没有看，但是也在查英文文档，也算变向练英语了吧。周四直播的内容来自The Economist,不算很长，讲述人口比例问题。学德语动力不太足，每天鹦鹉都处在癫狂状态。

新鲜知识/想法

  • 如果小米去做一把移液器？ 那价格肯定爆杀友商。也许可以自己做一个通用的点板提示装置，加装到普通移液器，提示滴板进度。

博客装修

花一个晚上大致研究了一下git submodule。现在主题和发布页文件夹有了单独的仓库，以submodule的方式与主仓库关联。主题的代码没有想象中的复杂，逻辑已经大致摸清。有时间就可以开工了。

折腾设备

  • Fedora server 和workstation版本采用一样的图形版安装程序。与Ubuntu server相比，安装程序更简单。自带简单的web console,页面设计赏心悦目。功能较少，但是可以满足基本需求（有了CLI就有了全世界）。

  • UART CLI 用的小米路由器自带UART串口，想试试用更电传打字机的方式连接命令行。目前只能接收，但是发送不了，hardware/software flow control也都关了，暂时不知道为什么。

  • DJI Phantom 4 电池 高危操作，切勿模仿 多年前的一次飞行后暴毙的电池，拆解研究一下。四块电池串联，内阻80Ω，应该没救了。电路板做工中规中矩，没有专门的散热措施。

  • 小米盒子4s 外壳没有螺丝，一圈全是卡扣，非常难拆，这很小米。主板上没有发现UART，但是根据网上资料存在刷机跳线。目前没有需求，设备继续吃灰。

读书动力并不太-强—啊——

  • 《spiral》 日文歌普遍比较热血。以前只听过米津玄師，这次是别人推荐的，比较豪庭,虽然歌词一点听不懂。

碎碎念

被制裁了

和朋友吐槽旅行计划泡汤的事。可能也是我吐槽这些他不知道怎么回比较好吧，他开始开导我。接下来就是大段的语音，开始说理，说明一个人去旅游也很好。二十分钟下来，我在吐槽，并且表示我只是想找个人出去见见太阳，他想开导我，但是给我的第一感受是想法被反驳。这次算是感受到在吐槽的时候别人一本正经开始分析有多让人无欲了。我觉得，以后，日常交流还是少点思辨比较好，我深刻反思🫠
